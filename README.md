# OMS

## 一、使用方法

>  $ git clone https://github.com/xytroot/OMS.git

并在src/main/resources/db.properties文件中修改数据库连接信息



## 二、技术栈

项目管理：Maven

前端：bootstrap、jsp、ajax、JavaScript、jQuery

后端：SSM(Spring + SpringMVC + MyBatis)框架

## 三、开发环境

IDE：IntelliJ IDEA Ultimate 2020.2.2

操作系统：Windows 10 专业版

开发语言：Java

数据库：MySQL 5.7

数据库连接池：c3p0

服务器：Tomcat 9.0.39
